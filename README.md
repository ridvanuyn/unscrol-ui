# Unscrol – Website

Static website for the **Unscrol** digital wellness app.  
Includes landing page, support center, and privacy policy.

## Pages

| Page | Path | Description |
|------|------|-------------|
| Home | `/` | App landing page |
| Support | `/support.html` | FAQ & contact information |
| Privacy | `/privacy.html` | Privacy policy |

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings** → **Pages**.
3. Under "Source", select **Deploy from a branch**.
4. Choose `main` branch and `/ (root)` folder.
5. Save — your site will be live at `https://<username>.github.io/unscrol-ui/`.

## Custom Domain (Optional)

To use a custom domain (e.g. `unscrol.app`):

1. Add a `CNAME` file to the repo root containing your domain.
2. Configure DNS records to point to GitHub Pages.
3. Enable "Enforce HTTPS" in the Pages settings.

## URLs for App Store / Google Play

After deploying, use these URLs:

- **Support URL:** `https://<username>.github.io/unscrol-ui/support.html`
- **Privacy URL:** `https://<username>.github.io/unscrol-ui/privacy.html`
